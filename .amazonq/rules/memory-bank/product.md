# ProfeFlow - Product Overview

## Project Purpose
ProfeFlow is a SaaS platform designed specifically for Chilean teachers to generate curriculum planning and evaluate student work using artificial intelligence. The application is aligned with Chile's national curriculum from the Ministry of Education (Mineduc).

## Core Value Proposition
- AI-powered curriculum planning generation aligned with Chilean educational standards
- Intelligent student work evaluation with constructive feedback
- Professional PDF export capabilities with flexible watermarking
- Tiered subscription model (FREE/PRO) for different usage needs

## Key Features

### AI Planning Generator
- Creates detailed curriculum plans by subject and grade level
- Fully aligned with Chilean Mineduc curriculum
- Generates learning objectives, activities, and assessments
- Class-by-class planning with customizable duration

### AI Evaluation Assistant
- Evaluates student work with constructive feedback
- Supports images and PDF documents
- Personalized feedback based on pedagogical criteria
- Portfolio analysis with comprehensive rubrics

### Professional Export System
- Downloads plans in professional PDF format
- Optional watermark based on user plan
- Print and presentation optimized formatting

### Flexible Plan System
- **FREE Plan**: 5 plans/month, 3 evaluations/month, PDF with watermark
- **PRO Plan**: Unlimited plans and evaluations, PDF without watermark

### Administrative Dashboard
- Complete dashboard for administrators
- Detailed usage metrics and analytics
- User and plan management
- Conversion and revenue statistics

## Target Users
- Chilean elementary and secondary school teachers
- Educational coordinators and administrators
- Curriculum specialists working with Mineduc standards

## Primary Use Cases
- Daily lesson planning aligned with national curriculum
- Student work evaluation and feedback generation
- Portfolio assessment for teacher evaluation processes
- Administrative oversight of platform usage and metrics
- Professional document generation for educational presentations
